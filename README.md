# Проект todoshka - Todo List Application

## Архитектура

Проект построен на Next.js (App Router), что позволяет сочетать фронтенд и бекенд в одном приложении.

### Фронтенд (Клиентская часть)

Находится в директории /src/app

- Собрано при помощи Next.js + TypeScript + React
- Используется компонент page.tsx с "use client" для клиентской логики
- Локальные состояния (useState) используются для форм, фильтров и состояния редактирования
- Запросы к серверу выполняются через стандартный fetch API

### Бекенд (Серверная часть)

Находится в директории /src/app/api

- Реализован с помощью Next.js API Routes
- Сервер работает на порту 3000
- Маршруты API организованы по папкам:
  - api/todos/route.ts – обработка GET (получить все заметки) и POST (создать заметку)
  - api/todos/[id]/route.ts – обработка PATCH (обновить заметку) и DELETE (удалить заметку)
  - api/todos/clear-completed/route.ts – удаление всех выполненных заметок
- Для работы с конкретными записями используется динамическая маршрутизация

### База данных

Проект использует PostgreSQL (совместима с Vercel Postgres для деплоя)

- Работа с БД через Prisma ORM (безопасность и удобные запросы)
- Схема базы данных описана в prisma/schema.prisma
- Функции работы с БД вынесены в src/lib/prisma.ts
- Подключение к БД через переменную окружения DATABASE_URL

### Стилизация

Глобальные стили находятся в /src/app/globals.css:

- Серебряный фон (#c2c2c2)
- Рамка приложения (#FBCEB1)
- Акцентный цвет (#d35400)
- Все интерактивные элементы (чекбоксы, кнопки удаления, фильтры) стилизованы

## Инструкция по запуску!!!!!!

### Локальная разработка (PostgreSQL локально)

1. Скачать проект из репозитория любым удобным способом
2. Распаковать его в любое удобное место
3. Открыть папку todoshka в терминале
4. Установить PostgreSQL локально (если не установлен)
5. Создать базу данных с именем todoshka
6. В файле .env заменить DATABASE_URL на вашу локальную PostgreSQL строку подключения (пример: postgresql://USER:PASSWORD@localhost:5432/todoshka?schema=public)
7. Установить зависимости: npm install
8. Инициализировать базу данных: npx prisma migrate dev --name init
9. Запустить приложение в режиме разработки: npm run dev
10. Кликнуть по ссылке напротив Local: (обычно http://localhost:3000)
11. Вы восхитительны!
